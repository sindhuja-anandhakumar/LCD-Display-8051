# 8051 LCD Interfacing

This project demonstrates interfacing a 16×2 LCD with the 8051 microcontroller. The LCD displays a welcome message across two lines using Embedded C. The project was developed using Keil uVision and simulated in Proteus.

## Features
- 16×2 LCD interfacing
- Displays custom text
- Embedded C program
- Developed using Keil uVision
- Simulated in Proteus

## Folder Structure

- `code/` – Source code
- `keil/` – Keil project files
- `Proteus/` – Proteus simulation files
- `Images/` – Circuit and output screenshots

## Software Used

- Keil uVision
- Proteus 8 Professional

## Microcontroller

- AT89S52 / 8051
